About the Site

PapaDoble is an API for literary cocktails, notably those favored by Ernest Hemingway. The API is both RESTful and stateless, takes JSON encoded requests, and returns JSON responses. This site serves as a front end for the API and a place for the documentation to live.

I had the idea because I have always been a fan of Hemingway's work, and I have wanted for a while to try to implement an API. The vast majority of this code was written at the Fall 2014 MHacks, and I'm pretty happy with how is has turned out so far. The names of many of these cocktails, and a few of the recipes, come from Phillip Greene's To Have and Have Another, which I highly recommend. All recipe's are cited with a source if you're wondering where I got them.

If you have any questions, feel free to shoot me an email at packrat386@gmail.com. Happy Mixing!
